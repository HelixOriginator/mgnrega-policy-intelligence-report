This report is part of the **Policy Intelligence System** — a framework designed to evaluate, compare, and map central government schemes using data, scoring models, and cross-scheme analysis.

## 📊 What This Report Covers

### 1. Policy Snapshot
- Legal nature (Act, not just scheme)
- Ministry, structure, nationwide coverage
- Rights-based architecture

### 2. Core Guarantee
- 100 days employment guarantee  
- Demand-driven model  
- Legal obligation of the state  

### 3. Financial Architecture
- ₹86,000+ crore annual spend  
- Centre-State cost sharing  
- Wage vs material ratio  

### 4. Scale & Data
- 15.99 crore households registered  
- 290+ crore person-days generated  
- ~30% rural household coverage  

### 5. Implementation System
- Job card → demand → allocation → work → payment  
- Gram Panchayat execution model  
- NREGASoft MIS backbone  

### 6. Output vs Outcome Analysis
- Outputs: employment, assets  
- Outcomes: income stability, migration reduction  

### 7. Ground Reality
- Success: scale, transparency, safety net  
- Failure: wage delays, asset quality, state disparity

## 🔁 Overlap Intelligence (Key Insight Layer)

This report identifies how MGNREGA interacts with other schemes:

- PM-KISAN → income support  
- PMFBY → risk protection  
- DAY-NRLM → livelihood development  

### Core Insight:
India’s rural welfare system is **fragmented**, not unified.

Same households interact with multiple schemes  
→ but no integrated policy architecture exists

---

## ⚠️ System-Level Insights

- MGNREGA works as a **crisis absorber**, not a transformation engine  
- Strong where governance is strong, weak where institutions fail  
- Wage delays are the **single biggest structural flaw**  
- No national system to measure **long-term outcomes**
